Модели БД

В приложении используются следующие модели:

  • User - Пользователь сервиса
  • Salary - Зарплата
  • SalaryPeriod - Период, за который считается получение зарплаты
  • SalaryPeriodForUser - Период получения зарплаты для конкретного юзера
  • Waste - Траты
  • WastePeriod - Период за который считаются траты
  • WastePeriodForUser - Период трат для конкретного юзера
